Sobre el proyecto

Es mi primer portafolio que hago y seguramente en el futuro lo voy a seguir necesitando, para mostrar los proyectos que haga.

Mis funciones: Naturalmente yo diseñé y desarrollé a la vez.

Es algo un poco simple en el desarrollo pero si es un poco extenso, porque la idea es terminarlo con la información necesaria y mostrar a los interesados.

Herramientas utilizadas

HTML y CSS

Por ciertas limitaciones que puede tener GitHub Pages y para mostrar lo antes posible el portafolio, solo utilicé html y css, para la estructura, el texto y los estilos de las páginas.


GitHub y Git

Por la url en la que está, necesariamente tuve que usar GitHub, lo desarrolle aquí porque creo que es lo más viable para un primer portafolio. También usé lo básico de git, aunque mayormente uso la interface de Visual Studio Code.
