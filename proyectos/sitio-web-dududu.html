SITIO WEB DE DUDUDU

Creación de sitio web

Sobre el proyecto

DUDUDU es una agencia de diseño y desarrollo web. Recién creada y yo era el desarrollador, se necesitaba un sitio web para tener presencia en internet y probar lo que podíamos hacer.

  • Yo hice la parte de operativa y el contenido: Desarrollo del sitio web y redacción del contenido.
  • Otro equipo tomó las fotos y mejoró el diseño: Yo primero hice un diseño sobre la marcha, pero con el feedback se tuvo que hacer algunas modificaciones en diseño y se incluyó fotos.

Luego se añadió nuevas secciones y pequeños cambios.

Herramientas utilizadas

HTML y CSS

Usamos estas dos herramientas porque eran suficientemente completos para desarrollar el sitio web, tanto en su estructura como en los estilos.


Archivo htaccess

Se usó para el archivo 404 y para que los archivos no se muestren en el url con la parte final .html


SEO

Para el SEO se usó los meta-tag y el script de rich snippets en el head, ademas del correcto uso de las etiquetas de HTML.


Google Analytics y Hotjar

Implementamos estas dos herramientas para las estadísticas del sitio web.
